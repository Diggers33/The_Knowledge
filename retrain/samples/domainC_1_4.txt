=== Domain C: hyperspectral imaging for automated plastic waste sorting in circular economy facilities ===
=== Section 1_4: Ambition and innovation beyond the state of the art ===

1.3.1 We will produce a novel HSI system with a complete set of components adapted to the specific needs of plastic sorting. This system will be entirely based on the integration of photonic components and will be composed of the following main components:

- Several miniaturized spectrometers to cover the complete spectral range of interest.
- Optical components to distribute the light to the different spectrometers.
- An illumination system based on LEDs.
- High-speed electronics adapted to the high data flow generated by the spectrometers.
- A real-time data processing system.
- A real-time data visualization system.

The miniaturized spectrometers will be based on a novel 2D photodetector array. The dispersive element will be based on a multi-layer dielectric (MLD) coating. This coating is composed of alternating layers of materials with different refractive indexes (RIs). The number of layers is high enough to create an interference effect. This interference effect causes a specific wavelength to be reflected and all the other wavelengths to be transmitted, or vice versa. The wavelength that is filtered out is called the central wavelength and it depends on the thickness and refractive indexes of the layers. This interference effect can be used to create an optical filter. The wavelength that is filtered out can be modified by modifying the angle of incidence (AOI) of the light on the optical filter. The MLD coating will be applied on top of the photodetector array. The photodetector array will be based on CMOS technology. The photodetector array and the MLD coating will be optimized to achieve a spectral resolution of 8 nm. To cover the full NIR spectral range, we will use 5 spectrometers with independent optical paths. The optical path will be optimized to ensure that the AOI of the light on the MLD is sufficiently different for each of the spectrometers.

The illumination system will be based on LEDs. LEDs will be chosen to cover the full NIR spectral range. LEDs will be optimized to provide the highest optical power possible. The LEDs will be arranged to ensure a uniform illumination of the plastic objects. The LEDs will be divided into several groups. Each group will be controlled independently. The lighting system will be optimized to maximize the signal-to-noise ratio (SNR) of the HSI system. The lighting system will be controlled in real-time based on the results of the data processing system. This control will allow us to optimize the SNR.

The data processing system will be based on a set of algorithms able to extract the most relevant features from the HSI data. It will be optimized to run on the edge. The data processing system will allow us to extract the most relevant features from the HSI data. The data processing system will be composed of two parts: an edge part and a cloud part. The edge part will allow us to extract the most relevant features from the HSI data. The cloud part will allow us to perform more complex data processing.
